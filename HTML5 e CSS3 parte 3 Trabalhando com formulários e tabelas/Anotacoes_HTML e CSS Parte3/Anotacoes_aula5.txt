Qual propriedade do CSS usamos para aumentar um elemento proporcionalmente?

transform: scale()

Alternativa correta! Utilizamos a propriedade transform: scale() para aumentar um elemento proporcionalmente.

Definição e Uso
A transformpropriedade aplica uma transformação 2D ou 3D a um elemento. Essa propriedade permite girar, dimensionar, mover, inclinar etc., elementos.

Para entender melhor a propriedade transform, veja uma demonstração: https://www.w3schools.com/cssref/css3_pr_transform.asp

---------------------------------------------------------------------------------------------------------------------------------------------------------------------------
Utilizamos a propriedade transform: rotate() para "girar" um elemento em seu eixo.
